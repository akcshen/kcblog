# 3X-UI 面板管理脚本完全指南

本文档详细介绍了 **3X-UI Panel Management Script** 的所有功能选项、使用场景及注意事项。适用于基于 Ubuntu/Debian/CentOS 等系统的服务器管理。

> **当前状态示例**:
>
> - `Panel state`: Running (面板运行中)
> - `Start automatically`: Yes (已开启开机自启)
> - `xray state`: Running (核心运行中)

------

## 1. 基础控制

| 选项  | 功能名称        | 详细说明                                                    |
| ----- | --------------- | ----------------------------------------------------------- |
| **0** | **Exit Script** | **退出脚本**。结束当前管理界面，返回到 Linux 命令行提示符。 |

------

## 2. 安装与维护

| 选项  | 功能名称           | 详细说明                                                | 注意事项                                                   |
| ----- | ------------------ | ------------------------------------------------------- | ---------------------------------------------------------- |
| **1** | **Install**        | **全新安装**。在从未安装过 3X-UI 的服务器上执行此操作。 | 若已安装，请勿随意点击，以免覆盖配置。                     |
| **2** | **Update**         | **更新面板**。将 3X-UI 程序和核心升级到最新版本。       | 建议定期更新以修复漏洞。**更新前建议备份数据库**。         |
| **3** | **Update Menu**    | **更新菜单脚本**。仅刷新管理脚本本身，不更新面板核心。  | 当菜单显示异常或需要最新管理功能时使用。                   |
| **4** | **Legacy Version** | **切换旧版**。回退到旧版本或原版 X-UI。                 | 仅在新版出现严重兼容性故障时使用。                         |
| **5** | **Uninstall**      | **彻底卸载**。删除面板、核心、配置文件及数据库。        | ⚠️ **高危操作**：所有节点数据和设置将丢失，操作前务必备份！ |

------

## 3. 账户与配置重置

| 选项  | 功能名称                      | 详细说明                                                     | 是否需要重启                       |
| ----- | ----------------------------- | ------------------------------------------------------------ | ---------------------------------- |
| **6** | **Reset Username & Password** | **重置登录凭证**。修改面板的用户名和密码。                   | ❌ 否 (立即生效)                    |
| **7** | **Reset Web Base Path**       | **重置访问路径**。修改或清空面板 URL 的后缀路径 (如 `/secret`)。 | ✅ 是 (需重启面板)                  |
| **8** | **Reset Settings**            | **恢复出厂设置**。清空所有自定义配置，回到初始状态。         | ⚠️ **高危**：会清除用户数据，慎用！ |
| **9** | **Change Port**               | **修改面板端口**。更改网页管理的监听端口 (默认通常为 54321)。 | ✅ 是 (需重启面板)                  |

------

## 4. 信息查看

| 选项   | 功能名称                  | 详细说明                                                     |
| ------ | ------------------------- | ------------------------------------------------------------ |
| **10** | **View Current Settings** | **查看当前设置**。显示面板地址、端口、用户名、路径、数据库位置等关键信息。 |

------

## 5. 服务运行控制

| 选项   | 功能名称         | 详细说明                                               | 适用场景                                  |
| ------ | ---------------- | ------------------------------------------------------ | ----------------------------------------- |
| **11** | **Start**        | **启动面板**。手动启动 3X-UI 服务。                    | 服务意外停止时。                          |
| **12** | **Stop**         | **停止面板**。手动停止 3X-UI 服务。                    | 临时维护或调试时。                        |
| **13** | **Restart**      | **重启面板**。重启 3X-UI 网页管理服务。                | 修改面板端口、路径后生效。                |
| **14** | **Restart Xray** | **重启核心**。仅重启 Xray 流量核心，不影响网页面板。   | **添加/修改节点后必须执行**，使配置生效。 |
| **15** | **Check Status** | **检查状态**。查看面板和核心的运行状态及开机自启状态。 | 排查服务是否正常运行。                    |

------

## 6. 日志与自动化

| 选项   | 功能名称              | 详细说明                                                    |
| ------ | --------------------- | ----------------------------------------------------------- |
| **16** | **Logs Management**   | **日志管理**。查看面板日志、Xray 核心日志，或清空日志文件。 |
| **17** | **Enable Autostart**  | **开启自启**。设置服务器重启后自动运行面板和核心。          |
| **18** | **Disable Autostart** | **关闭自启**。取消开机自动运行。                            |

------

## 7. 安全与网络优化

| 选项   | 功能名称                       | 详细说明                                                     |
| ------ | ------------------------------ | ------------------------------------------------------------ |
| **19** | **SSL Certificate Management** | **SSL 证书管理**。自动申请 (Let's Encrypt)、续签或上传 HTTPS 证书。 |
| **20** | **Cloudflare SSL Certificate** | **CF 证书管理**。专门配置 Cloudflare 边缘证书。              |
| **21** | **IP Limit Management**        | **IP 限制管理**。限制每个用户允许同时在线的 IP 数量。        |
| **22** | **Firewall Management**        | **防火墙管理**。一键管理服务器防火墙 (如 UFW)，放行或阻断端口。 |
| **23** | **SSH Port Forwarding**        | **SSH 端口转发**。配置本地隧道映射。                         |

------

## 8. 系统性能与工具

| 选项   | 功能名称               | 详细说明                                            |
| ------ | ---------------------- | --------------------------------------------------- |
| **24** | **Enable BBR**         | **开启 BBR 加速**。启用 Google BBR 拥塞控制算法。   |
| **25** | **Update Geo Files**   | **更新地理库**。更新 `geosite.dat` 和 `geoip.dat`。 |
| **26** | **Speedtest by Ookla** | **网速测试**。在服务器端运行专业测速。              |

------

## 9. 常用操作速查表

| 我想做什么？        | 推荐选项            | 备注                       |
| ------------------- | ------------------- | -------------------------- |
| **忘了密码/端口**   | `10`                | 查看当前设置               |
| **修改密码**        | `6`                 | 立即生效                   |
| **刚添加了新节点**  | `14`                | 重启 Xray 核心使配置生效   |
| **服务器刚重装好**  | `24` -> `6` -> `19` | 开加速 -> 改密码 -> 配证书 |
| **节点连不上/报错** | `16`                | 查看日志分析原因           |
| **面板打不开**      | `15` -> `22`        | 查状态 -> 查防火墙         |
| **定期维护**        | `2` -> `25`         | 更新面板 -> 更新地理库     |

------

### 💡 温馨提示

- **备份习惯**：在进行 `Update` (更新) 或 `Uninstall` (卸载) 等操作前，建议先备份数据库文件 (通常位于 `/etc/x-ui/x-ui.db`)。
- **安全性**：生产环境中，务必修改默认端口 (`9`)，设置强密码 (`6`)，并配置 SSL 证书 (`19`)。
- **日志先行**：遇到任何连接问题，第一反应应该是查看日志 (`16`)，而不是盲目重启。
