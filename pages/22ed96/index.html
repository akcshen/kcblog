# node 版本的切换

一. 安装 node 版本管理工具n

sudo npm install -g n
1

二、安装你需要的版本的 node

sudo n <node 版本号>

// 例如：
sudo n 11.0.0
1
2
3
4

三、查看/切换 你需要的 node 版本 上下切换 回车选择 q 退出

n
1

四、其他命令

// 安装最新版本
sudo n latest
// 安装稳定版本
sudo n stable
// 删除某个版本
sudo n rm <版本号>
