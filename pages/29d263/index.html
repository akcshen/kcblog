# 介绍

快应用类似于微信小程序，他的目标就是各个大厂联合起来对标微信小程序的，但是开发体验感觉远不如小程序，没个平台都有自己独立的 api，并没有保持文档的完全同步，而且处于初期阶段，所以 bug 也是相对较多，本文的目的主要是记录自己在开发过程中遇到的一些问题，方便记忆，谨防再犯
