微信更新后不能使用默认浏览器打开网站

# 方法一：

步骤：

第一步：⌃ + ⌥ + ⌘ + R（Control + Option + Command + R）

第二步：勾选「使用默认浏览器打开链接」

搞定！

# 方法二：

如果方法一的快捷键没用，可以在首页搜索框里面输入：

:recover
1

再见，垃圾的自带浏览器
