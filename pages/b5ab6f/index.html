# 用mi-recovey开机

  • 先按住上音量键，不要放
  • 再插上电源
  • 等一会就进入了mi-recovery界面，确认一下，浅蓝光标是不是在Reboot上，是的话就不要动放一边，等个5分钟就自动重启了(原理是长时间不操作就自动选择当前选择项，第一个就是重启选项)
