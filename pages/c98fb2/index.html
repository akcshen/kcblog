Cloudflare免费节点搭建详细教程

# Cloudflare 免费节点搭建详细教程（已绑定域名版）

你的域名后台地址：https://cfkv.kcshen.cn/admin (opens new window)

# 🧱 一、准备工作

你已经完成：

  • ✅ 拥有域名 cfkv.kcshen.cn (opens new window)（尚无域名可到 DNSHE (opens new window) 免费注册）
  • ✅ 域名已接入 Cloudflare

接下来只需要完成 KV + Pages + 绑定 + 配置

# 🗄️ 二、创建 KV 存储（必须）

# 步骤：

  1. 登录 Cloudflare 控制台 (opens new window)
  2. 进入：Storage & Databases → KV
  3. 点击 Create Namespace
  4. 命名（例如）：cfkv

# 📦 三、创建 Pages 项目

# 上传 ZIP

  1. Workers & Pages → Create → Pages
  2. 选择 Upload assets
  3. 上传项目源码（main.zip）

# 构建设置

  • Build command: 留空
  • Output directory: /

# 🔐 四、配置环境变量（非常关键）

路径：Settings → Environment Variables

添加：

ADMIN = 你的后台密码

# 🔁 五、重新部署

路径：Deployments → Retry deployment

# 🔗 六、绑定 KV 命名空间

路径：Settings → Functions → KV Namespace Bindings

  • Variable name: KV
  • KV namespace: cfkv

# 🌐 七、绑定域名

cfkv.kcshen.cn (opens new window) → CNAME → xxx.pages.dev（Pages 项目分配的子域）
开启 Cloudflare 代理（橙色云）

# 🔑 八、访问后台

浏览器打开：https://cfkv.kcshen.cn/admin (opens new window)
